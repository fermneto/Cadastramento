# PersonaLog

A simple command-line person registration system written in C.

---

## Features

- Register new people
- Edit existing records
- Delete records
- List and search registered people

---

## Getting Started

### Prerequisites

- GCC compiler or any C-compatible compiler

### Compilation

```bash
gcc main.c -o cadastramento
```

### Usage

```bash
./cadastramento
```

Navigate through the menu using the keyboard to manage records.
